## 🧠 一些关于算法的个人浅见（仅供快速入门）

在我看来，**算法其实是对计算机思想的凝练。**  
比如：

- **分治归并** → 产生了归排、快排等算法；  
- **树形结构理解** → 发展出了 DFS 等算法；
- **对性能的极致要求** → 产生了线段树的懒思想等；
- **对于重叠子问题** → 想出了类似于数学归纳法的、通过对状态转移的理解、有着回溯型DFS之称的DP。

💬 顺带一提：这些思想在其他领域也非常有用。  
例如在前端开发中，通过理解决策树的结构，我们可以在“分叉”处创建公共可复用组件，将不同分支的差异作为 `props` 传入，从而实现工程的**松耦合**以及**代码高复用率**。
